# Presentation Page

Open the presentation page in the browser:

- Local: `http://127.0.0.1/presentation`
- Deployed: `<your-base-url>/presentation`

What it includes:

- Slide-style browser presentation
- Sidebar slide navigator
- Previous / next controls
- Keyboard support with arrow keys

Before final submission:

- Replace the team-member placeholders
- Update the contribution statement with real names
- Adjust wording if your instructor wants a shorter or longer deck
