Project Motivation

Problem and Goal

We wanted a course project that feels realistic enough to demonstrate actual business workflows, but still stays within the scope of a database class. The focus is a clear database-backed system, not an oversized software engineering build.

Scope Manageable
Focus Database + UI
Requirement Full CRUD
Problem
Why this system is needed
Portfolio data spans accounts, assets, transactions, and risk rules, so users need one place to view it clearly.
Using manual SQL tools alone is not enough for a strong final demo because instructors need to see user-driven operations.
The project must show meaningful database behavior without becoming too broad for a semester course.
Goal
What we set out to build
Build a complete MySQL-backed application with a clean web interface and realistic financial entities.
Support portfolio monitoring, transaction handling, and risk management as core business workflows.
Demonstrate select, insert, update, and delete through the interface in a way that is easy to present.

Course Requirement

Database Operations Through the Interface

Required Operation How Our System Supports It Example Interface Pages
Select / Query Users can view portfolio summary, holdings, transactions, risk alerts, and reference data directly from the application. Dashboard, Holdings, Transactions, Risk, Master Data
Insert / Create Users can add new trades, assets, accounts, counterparties, users, and risk limits through form-based actions. Trading, Transactions, Risk, Master Data
Update / Modify Users can edit user records, risk limits, transaction information, and selected asset values from the interface. Transactions, Risk, Master Data
Delete / Remove Users can cancel transactions, close accounts, delist assets, and deactivate records without using SQL tools manually. Transactions, Master Data

This slide directly matches the course requirement that all four database operations must be performed through the application/interface rather than entered manually in MySQL Workbench or SQL tools.
